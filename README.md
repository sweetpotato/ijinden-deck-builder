# イジンデッキ

## これは何？

イジンデンのデッキレシピを作成するアプリです。

GitHub Pages でアプリを公開しています。公開ページは次の URL です。

https://sweetpotato.github.io/ijinden-deck-builder/

## 特徴

- デッキ枚数の上限なし
- デッキのカード枚数はカード名ごとに数字で表示
- メインデッキとサイドデッキを別個に管理可能
- デッキのカードの並びは種類、レベル、色、魔力コスト、エキスパンション順
- 作成したデッキレシピを共有リンクで他の人や自分の他の端末と共有可
- 作成したデッキレシピを「マイデッキ」として端末のブラウザに保存可能
- 作成したデッキで手札のシミュレーションが可能

## 未対応機能

- **自前の画像保存** -- OS のスクリーンショット機能で保存願います。
- **並び順の変更** -- ご面倒ですがこのまま使用ください。
- **PSR カード** -- 対応する SR のカードを使用ください。
