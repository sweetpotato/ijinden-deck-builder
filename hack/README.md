# 難読化

## 手順

1. 元画像を収集し、拡張子を除く名前を R-001 形式にする。
2. 前項を幅 110px x 高さ 154px に縮小した .png 画像にし、適当なディレクトリAに保管する。
3. 前項とは別のディレクトリBを作成する。
4. Aのディレクトリを pixzle-shuffle.sh にかけ、配下の画像を変換してBに保存する。
5. Bの画像を .webp に変換する。
6. .webp 画像を public/images にコミットする。

2と6には適当な画像編集ソフトを使えばいい。今は [Caesium Image Compressor](https://saerasoft.com/caesium/) を使用している。

pixzle-restore.sh は復元チェックに使う。

## パラメータ

- bs=22 -- ブロックサイズ。画像はこのピクセル数のタテヨコの正方形に分割される。縮小した画像のタテヨコの公約数であればいい。今の値は110と154の最大公約数である。
- seed=6880780254253071 -- 整数の乱数シード。MAX_SAFE_INTEGER (2^53-1) 以下であればいい。今の値は 2^52 以上 2^53-1 以下からランダムに選択した。
